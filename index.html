What We Collect

Full transparency about desktop app analytics

The desktop app includes opt-in anonymous usage analytics, disabled by default. You are asked once on first launch and can change your choice any time in Settings.

What we collect

  • Event types (app started, search completed, upload completed)
  • Result counts and durations (no content)
  • OS version and app version
  • Anonymous install ID (random UUID, not tied to you)

What we never collect

  • Document content or file names
  • Search queries or results text
  • File paths or folder names
  • IP addresses or personal information

Example event payload

{
  "event": "search.completed",
  "ts": "2026-02-20T10:15:30Z",
  "session": "a1b2c3d4e5f6",
  "install_id": "79e0d947...",
  "app_version": "2.4.8",
  "os": "Windows 10",
  "properties": {
    "result_count": 12,
    "duration_ms": 250
  }
}

Every event is logged locally. Open Settings > Usage Analytics > View Event Log to inspect exactly what has been recorded.
